# BJT 分析仪 (Model S) - 全自动双极性结型晶体管测试系统

![Version](https://img.shields.io/badge/Version-1.0-blue.svg)
![Python](https://img.shields.io/badge/Python-3.9+-green.svg)
![React](https://img.shields.io/badge/React-18-61dafb.svg)

BJT 分析仪 (Model S) 是一套专为双极性结型晶体管（BJT）设计的全自动化参数测试与特性评估系统。系统基于 Instruments Playground (IP-SDK) 底层硬件驱动，结合现代 Web 技术，为用户提供高精度、高可用性的测试体验。

## 🌟 核心功能

*   **自动极性识别**：插入器件后，系统自动判定晶体管为 **NPN** 或 **PNP** 类型。
*   **高精度参数测量**：
    *   实时显示：基极电流 ($I_B$)、集电极电流 ($I_C$)、电流放大倍数 ($\beta$)。
    *   测量精度：$V_{BE}$ 和 $V_{CE(sat)}$ 测量精度优于 ±0.5mV。
*   **全自动特性曲线扫描**：
    *   自动绘制 $I_C-V_{CE}$ 输出特性曲线族。
    *   支持高度可编程的扫描参数（起始 $I_B$、步进、阶数及 $V_{CE}$ 范围）。
*   **完善的硬件保护**：内置软件与硬件联动的 **过流保护 (OCP)**，超出设定阈值时瞬间切断激励源。
*   **ISO17025 标准报告**：测试完成后，可一键生成包含数据图表、不确定度声明的专业 Word/PDF 报告。
*   **本地数据管理**：内置 SQLite 数据库，自动归档每一次的历史测试数据。

## 📂 目录结构说明

系统采用前后端分离架构，代码结构清晰：

*   `backend/`：**核心驱动与服务层**。基于 FastAPI 构建，包含与底层硬件通信的 `hardware_tester.py` 驱动模块以及用于存储历史记录的数据库模块。
*   `frontend/`：**用户交互界面层**。基于 React + Vite + ECharts 构建，提供流畅的暗色模式实时数据大屏。
*   `docs/`：**技术文档库**。包含系统架构设计、硬件电路原理说明、操作手册及测试用例集。
*   `scripts/`：**自动化工具**。包含基于 `python-docx` 的专业测试报告生成脚本。

## 🚀 快速启动指南

### 环境要求
*   **操作系统**：Windows / macOS / Linux (树莓派)
*   **软件依赖**：Python 3.9+，Node.js 18+
*   **硬件依赖**：雨珠 (YZ) 系列测试仪硬件及配套 IP-SDK 环境

### 步骤 1：启动后端服务

1. 进入 `backend` 目录：
   ```bash
   cd backend
   ```
2. 安装 Python 依赖：
   ```bash
   pip install -r requirements.txt
   ```
3. 运行 FastAPI 服务：
   ```bash
   python app/main.py
   ```
   *服务将运行在 `http://0.0.0.0:8000`*

### 步骤 2：启动前端控制台

1. 开启一个新的终端窗口，进入 `frontend` 目录：
   ```bash
   cd frontend
   ```
2. 安装 Node 依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm run dev
   ```
4. 在浏览器中访问控制台地址（通常为 `http://localhost:5173`）。

## 📖 使用说明

1. **连接设备**：确保雨珠硬件已通过 USB 连接至电脑，系统左上角将显示绿色的 `● 已连接` 状态。
2. **放置器件**：将待测 BJT 晶体管正确插入测试座 (B、C、E 引脚对齐)。
3. **参数配置**：在左侧面板设置 $I_B$ 的起始值、步进、曲线阶数，以及安全的限流阈值。
4. **开始测试**：点击 **[▶ 开始测试]**。系统会自动识别器件，随后主屏幕的 ECharts 区域会实时、逐点绘制出特性曲线。
5. **导出报告**：测试完毕后，点击左下角的 **导出 PDF** 或 **导出 CSV**，即可获得标准化的测试结果报告。

*(详细操作步骤请参阅 `docs/04_Operation_Manual.md`)*

## 🔧 常见问题 (FAQ)

**Q: 为什么点击开始后提示“过流保护触发”？**
A: 请检查是否设定了过低的 `Ic 限流` 参数，或者晶体管的 C、E 极发生短路。系统为了保护内部运放，主动切断了输出。

**Q: 为什么报告无法生成？**
A: 请确保您已经完成了至少一次成功的完整测试，系统内部的 `scripts/generate_report.py` 依赖 SQLite 数据库中存在的最新测试记录。
