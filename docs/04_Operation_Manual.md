# BJT 分析仪 (Model S) 操作手册

## 1. 系统连接
1. 将 BJT 测试仪硬件通过 USB 连接至电脑。
2. 启动服务端：在终端运行 `python backend/app/main.py`。
3. 启动前端界面：在浏览器访问 `http://localhost:5173`。

## 2. 界面参数配置
在左侧控制面板中设置测试参数：
- **串口**：选择正确的设备路径 (默认自动识别 YZ 前缀设备)。
- **Ib 起始值 (μA)**：设定基极电流扫描的起点。
- **Ib 步进 (μA)**：每条曲线之间基极电流的增加量。
- **阶数**：共绘制多少条特性曲线 (例如 5 阶则绘制 5 条曲线)。
- **VCE 最大值 (V)**：集电极电压的扫描上限。
- **Ic 限流 (mA)**：安全保护阈值，当实际 $Ic$ 超过此值时自动停止测试。

## 3. 开始测试
1. 将待测晶体管正确插入测试座。
2. 点击 **[▶ 开始测试]** 黄色按钮。
3. 系统将首先执行器件识别（显示在右上角状态栏），随后进入自动扫描。
4. 主界面右侧的 ECharts 区域将实时绘制 $Ic-VCE$ 特性曲线。
5. 顶部的 5 个仪表卡片会实时更新当前的电压、电流及 $\beta$ 放大倍数。

## 4. 报告与数据导出
- **导出 CSV**：点击后将原始电压电流数据下载为 CSV 格式，便于在 Excel 中进一步分析。
- **导出 PDF**：点击后系统会调用内部 `generate_report.py` 引擎，生成符合 ISO17025 规范的测试报告。
