# BJT 测试系统硬件电路原理图说明

## 1. 核心测试拓扑 (基于差分采样)

为了满足高精度采样（±0.1%）和大范围扫描（Vce 0-50V, Ic 0-10A），本系统设计了外围功率级。

### 1.1 基极激励与测量 (Base Circuit)
- **激励源**：IP-SDK AWG1 输出 0-5V，经过精密恒流源电路转换为 1μA - 100mA 可编程基极电流。
- **采样**：基极回路上串联高精度取样电阻 $Rb_{sense}$ (0.1%精度)。
- **测量**：通过差分放大器 (如 INA282) 读取电阻两端压降 $Vb_{sense}$，送入 IP-SDK 的示波器 CH3。同时 CH1 测量基极对地电压 $V_B$。

### 1.2 集电极激励与测量 (Collector Circuit)
- **激励源**：IP-SDK AWG2 输出 0-5V，经大功率线性放大级 (OPA541 或分立晶体管对) 放大10倍，提供最高 50V 的 $V_{CC}$。
- **采样**：集电极回路上串联低阻值高精度检流电阻 $Rc_{sense}$ (0.1%精度)。
- **测量**：通过高速差分放大器读取压降 $Vc_{sense}$，送入示波器 CH4。CH2 测量集电极对地电压 $V_C$。

## 2. 保护电路设计 (Protection)
- **过流保护 (OCP)**：硬件比较器实时监测 $Vc_{sense}$，若 $Ic > Ic_{limit}$，硬件闭锁切断功率级输出。
- **过压保护 (OVP)**：功率级输出端并联 TVS 瞬态抑制二极管。
- **过温保护 (OTP)**：功率放大器散热片上贴片 NTC 热敏电阻，超过 85℃ 触发关断。
- **ESD 防护**：所有暴露在外的测试端子 (B, C, E) 均对地跨接低结电容 ESD 保护芯片。

## 3. 自动极性识别 (NPN/PNP Auto-detect)
硬件内置继电器矩阵，由 IP-SDK 的 Digital IO (静态数字IO) 控制。
识别逻辑：
1. 施加小幅正向 $V_{BE}$ 和 $V_{CE}$，若 $I_c$ 显著大于0，识别为 NPN。
2. 若无电流，切换极性施加反向电压，若检测到反向 $I_c$，识别为 PNP。
3. 否则提示“器件未接入或损坏”。
